fun unsafe-total( action : () -> e a ) : a

Unsafe. This function calls a function and pretends it did not have any effect at all. Use with utmost care as it should not be used to dismiss user-defined effects that need a handler and can cause a segfault at runtime in such cases! Only use this to dismiss built-in effects like exn or io.
